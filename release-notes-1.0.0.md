# AIMETON Cloud Bridge 1.0.0

Initial public release under the AIMETON Cloud Bridge identity.

Highlights:

- desktop and mobile Yandex Disk explorer;
- Russian and English interface;
- image preview and file transfers;
- visible transfer progress;
- strict desktop HTTP/HTTPS CONNECT and SOCKS5 proxy modes;
- SecretStorage for OAuth and proxy-password values;
- automatic migration from the former `yandex-disk-explorer` plugin ID.

See `MIGRATION.md` before replacing an existing private installation.
