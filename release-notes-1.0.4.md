# AIMETON Cloud Bridge 1.0.4

This maintenance release fixes operation error banners on mobile devices.

## Fixed

- Added a visible close button to the operation/error banner.
- Error banners can be dismissed by tapping the banner itself.
- Failed operation banners automatically disappear after 12 seconds.
- Prevented download timeout and network error messages from remaining permanently on screen.

The network, upload, download, preview, localization, theme, and proxy logic is unchanged.
