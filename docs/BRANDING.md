## Yandex Disk references

The project name intentionally does not contain “Yandex Disk”. References to Yandex Disk are descriptive: they identify the external service with which the plugin interoperates.

Before public catalog submission, review the current Yandex Disk API and brand terms. If an official service logo or attribution is required:

- obtain the current approved asset from the rights holder;
- do not redraw, recolor, crop, animate, or merge it with the AIMETON identity unless explicitly allowed;
- do not use it as the plugin's own logo;
- keep an independent-project disclaimer nearby where appropriate.

No third-party logo file is included in this repository package.

## Obsidian references

The project may be described as a community plugin compatible with Obsidian. It must not imply official status and must not use Obsidian's logo as the product logo without permission.

## Repository review

Before every public release, search the repository and Git history for:

- OAuth tokens beginning with `y0_`;
- proxy URLs containing credentials;
- personal file paths;
- private Yandex Disk download or preview URLs;
- screenshots exposing account or file information.
